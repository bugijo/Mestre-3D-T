# Mestre 3D&T – Painel do Mestre (Web/PWA)

## Visão Geral
Aplicativo Web Progressivo (PWA) para mestres de 3D&T que desejam organizar campanha, NPCs, encontros e som local em poucos toques.
**Nota de Auditoria (16/12/2025):** O projeto original Android foi descontinuado/perdido. O foco atual é a versão Web.

## Objetivos
- **Organização**: Gerenciar campanhas, arcos, cenas, NPCs e encontros.
- **Assistência em Sessão**: Painel rápido com gatilhos, NPCs da cena, encontro com ajustes de PV/PM e notas rápidas.
- **Áudio**: Painel de som com trilha e efeitos locais (Web Audio API).
- **Tecnologia**: TypeScript, Vite, React (Planejado), Tailwind CSS, IndexedDB (persistência local).
- **Offline First**: Funciona offline como PWA, com sincronização manual opcional via Supabase.
